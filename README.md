# TodoList (React + Vite)

This project is a simple to-do list built using React and Vite.

## Features:
- Add, edit, and delete tasks
- Local storage support
- Responsive UI

## Tech Stack:
- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [ESLint](https://eslint.org/)
- [SWC](https://swc.rs/)
